## Usage

To lint specs with swagger-node: 
```
npm run verify
```

To edit specs with swagger-editor:
```
npm run edit
``` 

To run the mock server(s) in mock mode:
```
npm run mock
```

To run the API server(s):
```
npm run serve
```

To run documentation site locally:
```
mkdocs serve
```

To deploy docs to gh-pages:
```
mkdocs gh-deploy
```
